# مشروع Tetrixa - نظام متكامل للإنتاج

هذا المشروع مجهز بالكامل للعمل في بيئة إنتاج آمنة ومستقرة. إذا كنت تنوي نقل المشروع إلى جهازك الخاص، فيرجى اتباع الإرشادات المتوفرة "داخل المشروع" أدناه.

## 🚀 البدء السريع (محلياً)

1.  **إعداد البيئة:** راجع الملف [ANDROID_SETUP_GUIDE_AR.md](ANDROID_SETUP_GUIDE_AR.md) لتثبيت SDK و NDK والمحاكي على جهازك.
2.  **إعداد التوقيع:**
    *   انسخ ملف `keystore.properties.example` باسم `keystore.properties`.
    *   املأ بيانات مفتاح التوقيع الخاص بك داخل الملف الجديد.
3.  **بناء التطبيق:** استخدم سكريبت الأتمتة المدمج:
    ```bash
    ./automation_build.sh
    ```

## 🛠 التكوينات المتقدمة

*   **R8 / ProGuard:** مفعل بالكامل لتصغير حجم التطبيق وتعميته برمجياً.
*   **التوقيع:** يعتمد النظام على ملفات الخصائص الخارجية لضمان عدم تسرب كلمات المرور.
*   **التوافق:** المشروع مهيأ للعمل حتى في البيئات التي لا تدعم التسريع العتادي.

## 📁 هيكل الأدوات المدمجة

*   `ANDROID_SETUP_GUIDE_AR.md`: الدليل الفني الشامل للإعداد.
*   `SPLASH_LOGO_GUIDE_AR.md`: دليل تغيير شعار شاشة البدء.
*   `automation_build.sh`: أداة بناء الإنتاج بضغطة واحدة.
*   `keystore.properties.example`: نموذج إعدادات الأمان.
